# 🧠 Human vs AI Text Classification

## 📌 Overview

This project solves the problem of detecting whether a text is written by a human or generated by AI.

The model was trained on a Kaggle competition dataset and achieved near top-1 performance.

---

## 📊 Dataset

Dataset is available on Kaggle:

https://www.kaggle.com/competitions/human-vs-ai-text-classification-feb2024

⚠️ Due to GitHub limitations, the dataset is not included in this repository.

---

## 🏆 Results

* My score: **0.99979**
* 1st place: 0.99991
* 2nd place: 0.99872

This solution outperforms most leaderboard participants.

---

## 🧠 Model Architecture

* Embedding (128)
* Conv1D (feature extraction)
* Bidirectional LSTM (context understanding)
* Global Max Pooling
* Dropout (0.5)
* Dense (sigmoid)

## 🚀 How to Run

### 1. Install dependencies

```
pip install -r requirements.txt
```

### 2. Download dataset from Kaggle

Place files into:

```
data/train.csv
data/test.csv
```

### 3. Train model

```
python src/train.py
```
