## 📌 Overview

This project solves the problem of detecting whether a text is written by a human or generated by AI.

Built using deep learning techniques and trained on a Kaggle competition dataset.

## 🧠 Model Architecture

* Embedding Layer (128-dim)
* Conv1D (feature extraction)
* Bidirectional LSTM (context understanding)
* Global Max Pooling
* Dropout (0.5)
* Dense (sigmoid)

## 🚀 How to Run

### 1. Install dependencies

```
pip install -r requirements.txt
```

### 2. Download dataset from Kaggle

Place files into:

```
data/train.csv
```

### 3. Train model

```
python src/train.py
```

### 4. Run inference

```
python src/predict.py
```
